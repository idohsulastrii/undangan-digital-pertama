# 🕊️ Prototipe Undangan Digital Modern

Proyek ini adalah riset awal untuk pembuatan jasa website undangan virtual dan katalog toko digital dengan konsep estetik, modern, dan berlayer (*glassmorphism*).

## ✨ Fitur Utama
- **Desain Responsif:** Tampilan khusus *mobile-first* (nyaman dibuka di HP tamu).
- **Efek Layer & Transparansi:** Menggunakan *backdrop-blur* dari Tailwind CSS.
- **Integrasi Google Maps:** Tombol navigasi langsung ke lokasi acara.
- **Amplop Digital Ringkas:** Tombol salin otomatis untuk nomor rekening tanpa ribet.

## 🚀 Rencana Pengembangan Jasa
- [ ] Membuat 3 variasi tema visual (*Earth Tone*, *Sage Green*, *Elegant Dark*).
- [ ] Uji coba integrasi formulir RSVP simpel menggunakan WhatsApp link.
- [ ] Konsep pemasaran jasa lewat konten visual (TikTok & Instagram Reels).
- [ ] Penggunaan sistem *Subdomain* khusus untuk klien (`nama-klien.brandkamu.com`).
